# Taj Unisex Beauty Salon

Premium website for **Taj Unisex Beauty Salon** — Sector 15, Panchkula, Haryana.

## Features

- Full-screen hero with luxury salon branding
- Services, gallery, and experience sections with premium imagery
- Online appointment booking with WhatsApp confirmation
- Mobile-responsive design with floating book button
- Contact section with location details
